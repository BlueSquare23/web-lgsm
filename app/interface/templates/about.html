About Web-LGSM

A simple web interface for managing LGSM game servers

Project Overview

The main idea behind this project is to provide a simple web interface to an otherwise CLI-only tool. This is a hobby / passion project, written mainly because not all of my friends are familiar with using SSH. Nonetheless, I wanted to give them access to stop, start, monitor, and generally administer our game servers.

This project exists to make server management more accessible and user-friendly.

Disclaimer

This project is NOT associated with the official LGSM project .

We are simply fans of their game server manager scripts and wanted to make them more accessible.
