About the Web-LGSM

The main idea behind this project is to provide a simple web interface to an otherwise CLI only tool. This is a hobby / passion project, written mainly because not all of my friends are familiar with using SSH. Nonetheless, I wanted to give them access to stop, start, monitor, and generally admin our game servers.

This project is NOT associated with the official LGSM project in anyway. We're simply fans of their game server manager scripts and wanted to make them easier to use and accessible to even more people!
